
# Frontend (React)

## Como rodar o frontend

**Pré-requisitos:**
- Node.js 20+
- npm

**Configuração local:**
```bash
cd frontend
npm install
npm start
```
O app estará disponível em http://localhost:3000/

**Com Docker:**
```bash
docker compose up --build
```
O frontend estará disponível em http://localhost:3000/
